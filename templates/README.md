# 템플릿 가이드

## thesis-template.docx 만드는 방법

1. Microsoft Word를 열고 새 문서를 만듭니다.
2. 학교 제출 양식에 맞게 다음 스타일을 설정합니다:
   - **제목1**: 논문의 주요 장 제목 (예: 1. 서론, 2. 관련 연구)
   - **제목2**: 하위 제목 (예: 1.1 연구 배경)
   - **제목3**: 세부 하위 제목
   - **본문**: 일반 본문 텍스트
   - **인용**: 인용문 스타일
   - **참고문헌**: 참고문헌 항목 스타일

3. 페이지 여백, 글꼴, 줄 간격 등을 학교 양식에 맞게 설정합니다.

4. 파일 > 다른 이름으로 저장 > Word Template (.docx)로 저장합니다.

5. 이 디렉토리에 `thesis-template.docx`로 저장합니다.

## 현재 상태

현재 기본 템플릿이 없으므로, Pandoc은 기본 Word 스타일을 사용합니다.
학교 제출용 양식이 필요한 경우 위 가이드에 따라 템플릿을 만들어주세요.
